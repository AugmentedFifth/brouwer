# brouwer

A scripting language inspired by Haskell, Python, and Rust, in that order.

## Rust stuff

```bash
$ git clone https://github.com/AugmentedFifth/brouwer.git
$ cd brouwer/rs
$ cargo run --release input_file.bwr
```

## C++ stuff

```bash
$ git clone https://github.com/AugmentedFifth/brouwer.git
$ cd brouwer/cpp
$ cmake .
$ make
$ ./brouwer input_file.bwr
```
